- Контроль успеваемости и проведение экзаменов
- Выдача сертификатов

## 8. Методологии

### 8.1 IDEF0 (Integration DEFinition for Function Modeling)
- **Стандарт:** Р 50.1.028-2001
- **Назначение:** Функциональное моделирование
- **Элементы:** Входы, Управление, Выходы, Механизмы
- **Применение:** Описание бизнес-процессов летной школы

### 8.2 IDEF3 (Process Description Capture Method)
- **Назначение:** Моделирование последовательности действий
- **Элементы:** UOB, Точки принятия решений, Синхронизация
- **Применение:** Детальное описание workflow-процессов

### 8.3 DFD (Data Flow Diagram)
- **Нотация:** Гейн-Сарсон
- **Элементы:** Процессы, Хранилища, Внешние сущности, Потоки
- **Применение:** Описание потоков данных в системе

### 8.4 IDEF1X (Integration DEFinition for Information Modeling)
- **Назначение:** Моделирование данных
- **Элементы:** Сущности, Атрибуты, Связи
- **Применение:** Проектирование реляционной базы данных
